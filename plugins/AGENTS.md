# AGENTS: 런타임 플러그인 및 설정

게이트웨이의 요청/응답을 가공/제어하는 런타임 플러그인과 그 설정을 담습니다.

## 구성 요소

- `config.yaml`: 플러그인 활성화/순서/설정 값 정의
- 내장 플러그인 예시:
  - `pii_filter/`: PII 탐지/마스킹
  - `deny_filter/`: 차단 규칙
  - `regex_filter/`: 정규식 기반 필터링
  - `resource_filter/`: 리소스 접근 제어
  - `resources/`: 리소스 선언 샘플

## 사용 방법

1) `plugins/config.yaml`에서 필요한 플러그인을 활성화/순서를 지정합니다.
2) 게이트웨이 기동 시 플러그인이 로드되어 요청/응답 파이프라인에 적용됩니다.
3) 문제 발생 시 로그 레벨을 조정하고, 플러그인별 설정을 좁혀가며 원인을 파악하세요.

## 보안 권장 사항

- 기본 차단 정책을 우선 적용하고 허용 목록을 좁혀가세요.
- 민감정보는 마스킹/익명화 후 외부로 전달되도록 강제하세요.
